# User Management System

A desktop application for managing users and their connections, built with JavaFX, PostgreSQL, and Maven.

The application allows:
- Creating and managing user profiles
- Managing relationships between users
- Associating users with interests
- Administering data through a simple interface

## Database Setup (PostgreSQL)

Create a database (example):

```
cg_app
```

Run this SQL script:

```sql
CREATE TABLE users (
id INT PRIMARY KEY,
name TEXT NOT NULL,
username TEXT NOT NULL UNIQUE,
avatar TEXT NOT NULL
);

CREATE TABLE interests (
id INT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
label TEXT NOT NULL UNIQUE
);

CREATE TABLE connections (
user_id INT NOT NULL,
friend_id INT NOT NULL,
PRIMARY KEY (user_id, friend_id),
FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
FOREIGN KEY (friend_id) REFERENCES users(id) ON DELETE CASCADE,
CHECK (user_id < friend_id)
);

CREATE TABLE user_interests (
user_id INT NOT NULL,
interest_id INT NOT NULL,
PRIMARY KEY (user_id, interest_id),
FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
FOREIGN KEY (interest_id) REFERENCES interests(id) ON DELETE CASCADE
);
```

(Optional) Insert test users:


# Architecture Overview

### App.java
Entry point of the JavaFX application.

### SceneRouter
Manages navigation between scenes (Role Select, Admin, User).

### DbUserRepository
Handles all database interactions using JDBC.

### Controllers
- RoleSelectController
- AdminController
- UserController

Each controller manages UI logic and user interaction.
